				
■ネットワーク関連用語				
				
IP				
	IPってなんの略？＞Internet Protocol		
	どんなプロトコル？	＞データパケットがネットーワークを経由して正しい宛先に到達できるように、		
		　　　　　　データパケットをルーティングおよびアドレス指定するためのプロトコル		
	どこの層で動作するプロトコル？＞TCP／IPの階層モデルにおいてはインターネット層で動作するプロトコル		
	どうやって識別させてる？＞IPアドレスを各ノードに割り当てることで、各ノードを識別することができる		
		　　　　　　　　　このIPアドレスの宛先を確認することで、あるノードから別のノードへデータを送信することができる		
	どこのヘッダに含まれる？＞宛先情報については、IPヘッダに含まれる		
				
IPヘッダ	
　　　　　IPヘッダって何＞IP(Internet Protocol)におけるデーだ送受信単位であるIPデータグラム(パケット)の先頭にある、
　　　　　　　　　　　　　一定の形式に従って制御情報などを記載した部分のこと
		

【インターネット制御メッセージプロトコル】				

ICMP	
　　　　　ICMPってなんの略？	＞Internet Control Message Protocol		
	どんなプロトコル？	＞ネットワーク通信の問題を診断するためにネットワークデバイスが使用するネットワーク層のプロトコル
　　　　　ICMPは何に使用される？＞【Query】問い合わせ用のメッセージ。監視サーバーからこのメッセージを監視するノードに対して
　　　　　　　　　　　　　　　　　　送信し、通信状態を確認します。
　　　　　　　　　　　　　　　　　【Error】監視サーバーから監視するノードにテストパケットを送付した際、経路途中でパケットが破棄された場合に発せられるメッセージ。破棄の原因もふせて知らせてくれる。
　　　　監視サーバーってなんのこと＞


ICMPメッセージ　
　　　　　　　　ICMPってなんの略？	＞Internet Control Message Protocolメッセージ
　　　　　　　　メッセージってどう言うもの？＞タイプフィールドの性質によって種類は主に
　　　　　　　　　　　　　　　　　　　　　　　「Queryメッセージ」「Errorメッセージ」の２つに分けられる
　　　　　　　　
　　　　　　　　タイプフィールドって何？
　　　　　　　　Queryメッセージ＞IP通信の疎通状態の調査や特定のノードに対する情報の問い合わせを目的とした
　　　　　　　　　　　　　　　　　タイプのメッセージ群を指します。
　　　　　　　　Errorメッセージ＞IP通信において何らかの問題が発生したときに、経路途中のノードから送信元ホストへ
　　　　　　　　　　　　　　　　　エラー通知することを目的としたタイプのメッセージ群を指します。
　　　　　　　　ホストってどこ？＞コンピュータネットワークにおける接続されたコンピュータやデバイス（
　　　　　　　　　　　　　　　　　端末、周辺機器（サーバ、パソコン、スマートフォン、ルータ））の総称のこと
